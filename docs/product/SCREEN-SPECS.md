# Screen Specifications v0.1

## 全局移动端框架

底部导航：`命盘 / 问盘 / 档案 / 我的`。

命盘、报告、Agent顶部均显示当前 Profile。Profile Switcher 支持快速切换和新建档案。

## S01 出生资料

- 单页或最多两步，避免冗长 onboarding。
- 城市搜索自动填充时区和经纬度。
- 主按钮：`免费生成命盘`。
- 清晰展示时间准确度说明和隐私说明。

## S02 免费命盘

视觉基准：C1结构，使用B版整体设计语言并减少梦幻装饰。

内容顺序：

1. 标题与 Profile Switcher
2. 右上角 `美化 / 专业` 切换
3. 南印度命盘
4. 上升、月亮、命主星、当前大运
5. 核心摘要
6. `解锁完整报告` 主按钮
7. `免费问盘` 次按钮
8. 报告章节预览

### 美化模式

- 星座和行星使用统一图标。
- 降低宫位编号权重。
- 上升点清晰高亮。
- 默认只显示普通用户能理解的关键数据。
- 点击宫位以底部抽屉显示中文解释。

### 专业模式

- 宫位、行星度数、Nakshatra、Pada、宫主星。
- 分盘切换。
- Yoga、Shadbala、Ashtakavarga、Dasha 层级。
- 专业名词均可点击解释。
- 保持命盘本体准确，不能用生成图片代替数据渲染。

模式偏好按用户保存，切换不触发重新计算。

## S03 报告预览

视觉基准：B2。

- 每个章节是清晰独立Section，不做十个相同小卡片。
- 显示章节编号、图标、标题和一句免费预览。
- 预览句后使用渐隐和锁定提示，不生成假正文。
- 底部固定按钮：`解锁完整人生报告`。
- 页面仍提供进入免费Agent的入口。

## S04 购买确认

- 当前人物和头像/昵称。
- 报告章节清单。
- 附赠问盘次数。
- 预计生成时间。
- 总价和支付按钮。
- 订单幂等、防重复点击和失败恢复。

## S05 报告生成

- 显示真实章节状态，不使用假百分比。
- 显示根据监控估算的剩余时间。
- 告知可以离开，报告会保存到档案。
- 失败时标记具体章节并允许重试。

## S06 完整报告

- 原生网页长阅读，不提供PDF作为核心交付。
- 吸顶章节目录。
- 暖白阅读背景。
- 每条重要结论可展开查看盘面依据。
- 每章底部：`针对本章节继续提问`。
- 提供生成重点分享卡，而非导出整份报告。

## S07 AI问盘

视觉和信息架构基准：A3。

- 顶部 Profile / 多人物选择器。
- 明确显示依据：本命盘、当前大运、某报告章节等。
- 显示剩余问盘次数。
- 输入框上方横向快捷问题。
- 用户提交前显示本次消耗。
- 回答中提供盘面依据折叠区。
- 未购报告用户可看到相关 `获取完整报告` 按钮。
- 输入区适合拇指操作，保持在安全区上方。

## S08 档案

- 人物列表：昵称、关系、基础命盘状态、报告状态。
- 新建、编辑、归档。
- 删除前提示报告和对话影响。
- 支持选择两个档案进入关系分析。

## S09 分享卡

视觉基准：A4结构 + C4传播力。

模板：核心人格、事业天赋、感情模式、当前阶段。

允许内容：分享名、上升、月亮、命主星、关键词、一句摘要、品牌、真实二维码。

禁止内容：出生日期、出生时间、出生地点、订单、积分、完整专业数据。

生成前显示预览和隐私检查。中文文字与QR必须代码渲染，不使用图像模型直接生成最终成品。

## S10 我的

- 剩余问盘次数
- 购买记录
- 问盘包
- 邀请奖励
- 设置和隐私
- 删除账户与数据导出入口
