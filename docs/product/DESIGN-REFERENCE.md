# Design Reference v0.1

## 已选择的组合

- **整体系统：B** — 暖白、靛蓝、陶土、矿物绿；现代吠陀几何；中文阅读友好。
- **免费命盘：C1** — 保留其视觉吸引力和图标化命盘，但移除过度星空、莲花/宗教式轮廓和不准确装饰。
- **报告预览：B2** — 清晰章节分割和渐隐锁定。
- **Agent：A3** — 最清楚的上下文、回答、盘面依据、快捷问题和购买入口布局。
- **分享卡：A4 + C4** — A4的信息层次、C4的传播冲击力；最终使用代码渲染真实中文与二维码。

## 暂定视觉Token（不是最终品牌定档）

- 暖象牙：`#F6F1E8`
- 深靛蓝：`#14233A`
- 陶土：`#B8613C`
- 矿物绿：`#3F6E61`
- 克制金：`#C3A46D`
- 正文：`#20242C`
- 次级文字：`#6F7279`

报告阅读优先浅色；命盘可使用深色重点区域；Agent以A3的浅色信息架构为主。不要把所有页面统一做成黑底。

## 禁止方向

- 印度神像、寺庙、宗教图腾
- 莲花与曼陀罗堆叠
- 廉价黑金算命风
- 霓虹赛博星空
- 大量玻璃卡片和渐变
- 技术后台式密集表格作为普通模式
- 用ImageGen生成真实命盘、中文正文或最终二维码

## 实现要求

- 所有真实UI文字必须是HTML/原生组件。
- 真实命盘必须由结构化数据动态渲染为SVG/Canvas/DOM。
- 图标应来自统一矢量系统。
- 概念图仅用于层级、密度、配色与情绪参考。
- 在375px、390px、430px手机宽度验证。
- 支持安全区、动态字体、无障碍对比度和减少动态效果。
