# PRD — 中文吠陀占星 Web App v0.1

## 1. 产品定义

面向中文用户的移动优先吠陀占星产品。以确定性命盘计算为可信基础，以网页完整报告作为主要付费商品，以基于命盘的 AI 问盘作为免费体验、报告延伸和复购入口。

产品不是纯聊天机器人，也不是只给专业人士看的技术排盘工具。

## 2. 核心价值

- 普通用户：无需理解复杂术语，也能获得清晰、可追溯的命盘解读。
- 进阶用户：可切换专业模式查看宫位、度数、星宿、Pada、Yoga、Shadbala、Ashtakavarga、Dasha 与分盘数据。
- 付费用户：获得分章节的完整人生报告，并可从任一章节进入 Agent 追问。
- 多人场景：保存多个档案，并进行单人、双人及后续多人关系/合作分析。

## 3. 产品形态

- 第一版：响应式 PWA Web App。
- 中文优先，手机优先，同时兼容桌面浏览器。
- 后续可适配微信小程序或使用壳层封装原生 App。
- 核心排盘由现有 Python/FastAPI 引擎提供。

## 4. 商业模型

第一版不做订阅。

### 免费权益

- 创建和保存基础人物档案。
- Python 确定性基础排盘。
- 美化模式命盘。
- 上升、月亮、命主星、当前大运等关键数据。
- 一段核心摘要。
- 每个付费章节一句预览。
- 初始免费问盘次数。

### 一次性付费

`完整人生报告`：付款后才触发长报告生成，完成后永久保存在用户档案中，并附赠固定问盘次数。

报告章节建议：

1. 本命结构
2. 性格与天赋
3. 事业与财富
4. 感情与婚姻
5. 身心健康
6. 家庭与人际
7. 当前大运
8. 未来十二个月
9. 关键机会与风险
10. 完整盘面依据

### 问盘付费

- 免费用户也能使用 Agent，但次数有限。
- 普通单人问题消耗 1 次。
- 深度专项、双人或多人分析可以消耗更多次数。
- 报告购买附赠问盘次数。
- 可单独购买问盘包。
- 邀请有效新用户可获得问盘次数，设置月度上限与防刷机制。

价格和积分数值全部由后端配置，不写死在前端。

## 5. 成本控制原则

### 免费阶段

1. 执行低成本、确定性的 Python 排盘。
2. 保存结构化命盘 JSON。
3. 使用一次受限模型调用生成结构化预览 JSON：核心摘要、章节一句话预览、推荐问题。
4. 不生成完整报告，不生成被锁定章节正文。

### 购买之后

1. 复用已保存的出生资料与命盘 JSON。
2. 必要时重新校验排盘，但不重复无意义计算。
3. 按章节异步生成完整报告。
4. 保存每个章节状态、结果、模型与版本。
5. 失败章节可单独重试，避免重跑整份报告。

### Agent

按问题调用，只注入所需上下文：当前人物、命盘摘要、必要的 Dasha/Transit、相关报告章节和有限对话历史。

## 6. 核心用户对象

- User：账户。
- Profile：一个人物档案及出生资料。
- Chart：确定性命盘计算结果和版本。
- Preview：低成本免费摘要。
- Report：购买状态、生成状态和完整报告。
- ReportSection：单独生成、重试和阅读的章节。
- Conversation：单人或多人问盘会话。
- CreditLedger：不可变的积分/问盘次数账本。
- Order：报告或问盘包订单。
- Referral：邀请关系和奖励状态。

## 7. 多人物能力

- 用户可创建自己、伴侣、亲友等多个 Profile。
- 命盘、报告和 Agent 页面都允许切换当前人物。
- Agent 支持选择一个或多个 Profile。
- 双人模式优先支持感情、婚姻与合作分析。
- 多人模式属于后续增强功能，MVP可先完成数据模型和选择器，不一定首发完整解读。
- 页面必须始终明确显示当前分析对象，防止串档。

## 8. AI 输出原则

- 遵守仓库 `RULES.md` 防污染规则。
- 每个重要结论必须能够追溯到命盘依据。
- 免费用户的 Agent 回答可以出现自然的完整报告推荐，但不能故意给出无用答案逼迫购买。
- Agent 中的购买入口必须与当前人物绑定。
- 医疗、投资等高风险内容必须明确为参考性解读，不能替代专业意见。

## 9. MVP范围

### 必须有

- 登录或可迁移的游客身份
- 出生资料录入与地点/时区解析
- 多人物档案
- 免费命盘与低成本预览
- 美化/专业模式切换
- 报告付费墙与模拟支付状态
- 异步报告进度
- 网页报告阅读
- 单人 Agent 问盘
- 章节跳转问盘
- 问盘次数账本
- 分享卡预览与隐私规则

### 暂缓

- 订阅
- 完整多人群体关系算法
- 公开社区
- PDF报告
- 自动出生时间校正
- 复杂运营后台
- 真实小程序支付，直到 PWA 漏斗验证完成

## 10. 成功指标

- 出生资料完成率
- 免费命盘生成成功率与耗时
- 免费预览到报告购买点击率
- 购买完成率
- 报告生成成功率与平均耗时
- 免费问盘使用率
- 报告用户的问盘使用率
- 赠送次数用完后的问盘包购买率
- 有效邀请率
- 单次免费预览、完整报告和问盘的真实模型成本
